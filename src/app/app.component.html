Avaliação 1

Thaylla | Disciplinas

Intruções

  1. (10 pontos) O AppComponent deve apresentar um menu de navegação com links para os componentes solicitados abaixo.
  2. (5 pontos) O app.module deve ter configuradas as rotas para os componentes solicitados abaixo.
  3. (5 pontos) Um componente chamado “[SeuNome]Component” que apresente seus dados de identificação: nome completo, RA (registro de aluno ou matrícula) e link para o perfil do GitHub
  4. (20 pontos) Um service chamado “[SeuNome]DisciplinasService” que permita listar, incluir e excluir uma lista de disciplinas (código, nome, diaDaSemana, horario e ementa) já inicializada com as disciplinas cursadas por você neste semestre
  5. (20 pontos) Um componente chamado “[SeuNome]ListaComponent” que apresente ao usuário uma tabela com suas disciplinas mostrando código, nome, dia da semana e horário e um link para visualizar os detalhes da disciplina
    1. (10 pontos) Deve haver um formulário que permita incluir uma nova disciplina.
    2. (10 pontos) Deve haver um comando em cada linha da tabela para excluir a disciplina.
  6. (20 pontos) Um componente chamado “[SeuNome]DetalhesComponent” que apresente ao usuário os detalhes de uma disciplina a partir de um link do componente anterior
